# Система Управління Прокатом Велосипедів (Bike Rental Management System)

Цей проект є бекенд-системою для управління процесом прокату велосипедів. Він надає RESTful API для взаємодії з даними про користувачів, велосипеди, локації та транзакції прокату, а також функціонал для аналітики.


## Технології

Проект розроблено з використанням наступних технологій та бібліотек:

-   **Python**: Основна мова програмування.
-   **FastAPI**: Сучасний, швидкий (високопродуктивний) фреймворк для створення API на Python, заснований на стандартних підходах Python Type Hints.
-   **SQLAlchemy**: Потужний Object Relational Mapper (ORM) для взаємодії з реляційними базами даних.
-   **Alembic**: Інструмент для міграції баз даних, інтегрований з SQLAlchemy.
-   **Pydantic**: Бібліотека для валідації даних та управління налаштуваннями за допомогою Python type hints.
-   **Uvicorn**: Швидкий ASGI-сервер для запуску FastAPI-додатків.

## Сутності

Система оперує наступними основними сутностями:

-   **Велосипед (Bike)**: `id`, `brand`, `price_per_hour`, `location_id` (місцезнаходження велосипеда).
-   **Локація (Location)**: `id`, `name` (назва точки прокату).
-   **Користувач (User)**: `id`, `name`, `phone` (контактні дані).
-   **Прокат (Rental)**: `id`, `user_id`, `bike_id`, `from_time` (час початку), `to_time` (час закінчення).
-   **Знижки (Discounts)**: Деталі для реалізації системи знижок. (Поля можуть бути: `id`, `name`, `percentage`/`amount`, `start_date`, `end_date`, `bike_id`/`user_id`.)

## Функціонал

Проект реалізує наступний функціонал:

-   **CRUD-операції**: Повний набір операцій (створення, читання, оновлення, видалення) для всіх основних сутностей (Велосипеди, Локації, Користувачі, Прокати).
-   **Доступність велосипедів за локацією**: Можливість переглядати, які велосипеди доступні для прокату в конкретній точці.
-   **Топ-локації за прокатом**: Аналітичний звіт, що показує найбільш популярні локації на основі кількості здійснених прокатів.
-   **Прибуток за період**: Розрахунок загального прибутку сервісу за день або місяць.
-   **Велосипеди зі знижками**: Функціонал для відображення та управління велосипедами, на які діють знижки.

## Встановлення

Для встановлення та запуску проекту виконайте наступні кроки:

1.  **Клонуйте репозиторій:**
    ```bash
    git clone [https://github.com/](https://github.com/)[Sn4-0]/bike-rental-api.git
    cd bike-rental-api
    ```

2.  **Створіть та активуйте віртуальне середовище (рекомендовано):**
    ```bash
    python -m venv venv
    # Для Windows:
    .\venv\Scripts\activate
    # Для macOS/Linux:
    source venv/bin/activate
    ```

3.  **Встановіть залежності:**
    ```bash
    pip install -r requirements.txt
    ```
